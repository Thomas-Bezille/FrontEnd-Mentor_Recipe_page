## Help

1. Clone the repo
2. Activate the virtual environment

   ```bash
   source .env/Scripts/activate
   ```

3. Start the server

   ```bash
   py src/manage.py runserver
   ```

4. Go

   ```text
   http://127.0.0.1:8000/recipe/
   ```
